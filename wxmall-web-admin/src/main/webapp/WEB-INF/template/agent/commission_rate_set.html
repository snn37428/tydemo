帮助说明

  1. 直属上级佣金比例：分销商推荐买家购买后能拿到的佣金比例。
  2. 二级上级佣金比例：分销商推荐买家购买后，分销商的上级分销商能拿到的佣金比例
  3. 三级上级佣金比例：分销商推荐买家购买后，分销商上级的上级能拿到的佣金比例
  4. 启用分销内购后，分销商购买商品自己可以拿一级佣金，自己的上级拿二级佣金； 未启用时，分销商购买商品时自己是拿不到佣金的，自己的上级可以拿一级佣金。
